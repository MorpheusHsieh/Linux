9. 建立 Ramdisk

  編輯 grub.conf 

  # vi /etc/grub.conf

  找到 

  kernel /vmlinuz-2.6.9-42.0.10.EL ro root=/dev/VolGroup00/LogVol00

  並在後面加上 "ramdisk_size=131072"，所以最後應該長得很像

  kernel /vmlinuz-2.6.9-42.0.10.EL ro root=/dev/VolGroup00/LogVol00 ramdisk_size=131072

  然後儲存並離開 grub.conf 後重新開機

  # reboot

  重開機後，格式化 ramdisk 並 Mount ramdisk，這裡假設 ramdisk 的擁有者為 t1280

  # mke2fs -m 0 /dev/ram0
  # mkdir /home/t1280/ramdisk
  # mount /dev/ram0 /home/t1280/ramdisk
  # chown t1280.t1280 /home/t1280/ramdisk

  如果要在每次開機的時候啟動 ramdisk, 則編輯 /etc/rc.local 檔案

  # mke2fs -m 0 /dev/ram0
  # mount /dev/ram0 /home/t1280/ramdisk
  # chown t1280.t1280 /home/t1280/ramdisk


10. 將帳號 t1280 的權限提升

  設定 t1280 不用再鍵入密碼就可變成 root 

  # visudo
  
  找到下列文字
  # %wheel        ALL=(ALL)       NOPASSWD: ALL         //預設值
  
  修改成下列文字
  %wheel ALL=(ALL)       NOPASSWD: ALL                  //修改後
 
  修改 /etc/group，並修改內容
 
  # vi /etc/group
  
  wheel:x:10:root                                       //預設值
  wheel:x:10:root,t1280                                 //修改後
