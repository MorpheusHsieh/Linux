How to create a sudo user on Ubuntu 16.04


1.Add a username

    $ sudo adduser [username]

2.Use the usermod command to add the user to the sudo group.

	$ sudo usermod -aG sudo [username]

	
3.Test sudo access on new user account

	$ sudo ls -la /root
